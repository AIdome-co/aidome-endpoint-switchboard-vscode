# Welcome to LLM Endpoint Switchboard

The LLM Endpoint Switchboard helps enterprise teams manage AI coding assistant configurations from a single place.

## What it does

- **Detects** your installed AI assistants (Cline, Continue, Roo Code, Kilo Code, Codex, and more)
- **Creates** endpoint profiles pointing to your approved LLM gateway
- **Configures** each assistant automatically (Tier A) or guides you through setup (Tier B/C)
- **Verifies** that everything is working with a 7-step verification pipeline

## What it doesn't do

- ❌ It does NOT proxy, relay, or intercept any API calls
- ❌ It does NOT collect any telemetry or usage data
- ❌ It does NOT store secrets in plain text (uses VS Code SecretStorage)
