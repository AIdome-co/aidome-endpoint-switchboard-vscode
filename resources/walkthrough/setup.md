# Running the Setup Wizard

The setup wizard guides you through:

1. **Detecting** installed AI assistants
2. **Selecting** which assistants to configure
3. **Creating** an endpoint profile (base URL + API dialect + auth)
4. **Applying** configuration changes
5. **Verifying** connectivity

## Tips

- Tier A assistants (Continue, Cline, Roo Code, Kilo Code, Codex) are fully automated
- Tier B/C assistants provide guided instructions
- All changes can be undone via "AIdome: Reset Switchboard"
